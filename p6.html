    • #6 louie interaction data

    Description

    In this project, I collected data from my interactions with my cat, Louie. The interactions span from petting, snuggling, argueing, accidents, playing, and feeding. In this code. the color of the dots reprsent the diffent courses of action, and the columns represent the days of the week.

    Design Process

    In this code, I used the random command to represent the interactions, and designed the dots to group up in vertical lines to define the seperate days and data collected each day. I used the methods we learned previously, forLoops, and applied the idea of a stagger with no actual movement and the data changes each week and nothing is contant.

    Reflection

    When I started to code this data, I realized that I was coding opposite to my sketch. I ended up getting lost in the code and creating a different graph to represent my data. If given more consideration and mapping out the sequence of dots and colors, I would have been able to design my sketch accurately.
